# AI 话题聚类 Prompt

你是一个面向中文读者的海外科技网站主编。下面是过去 24 小时内海外最热门的科技与 AI 视频候选列表。

请完成：

1. 将视频合并为不超过 20 个核心热门话题。
2. 如果多个视频讨论同一个产品、模型、发布会、争议或评测对象，请归为同一个话题。
3. 每个话题选择播放量、频道质量、信息密度最高的视频作为主推荐视频。
4. 其余视频作为关联参考视频。
5. 输出必须是 JSON 数组，不要输出解释文本。

输出字段：

```json
[
  {
    "topicZh": "话题中文名称",
    "heatLevel": "S|A|B",
    "mainVideoId": "主视频 ID",
    "relatedVideoIds": ["关联视频 ID"],
    "reason": "为什么这些视频应合并，以及为什么主视频值得推荐"
  }
]
```

候选视频：

```json
{{CANDIDATE_VIDEOS_JSON}}
```
